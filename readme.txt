3D-модель дерева на OpenGL (>= 3.2). Реализованы геометрическое построение моделей,
инстанциирование, построение с элементом случайности, освещение по Фонгу,
свободная камера (управление тривиально: "W", "A", "S", "D" и мышь), реалистичные текстуры,
колыхание дерева (с элементом случайности, амплитуда зависит от толщины ветки).
